# 医学统计与 R 语言数据分析 (Medical Statistics & R Programming)

本项目记录了我在医学研究中应用 R 语言进行数据处理、统计分析及可视化的学习过程与实战代码。

---

## 📂 项目模块

### 1. 📊 临床数据清洗 (Data Cleaning)
- **缺失值处理**：针对临床采集数据中常见的空项进行清洗与插补。
- **变量转换**：将原始护理记录数据转化为可供统计分析的结构化因子（Factors）。

### 2. 🧪 统计分析实战 (Statistical Analysis)
- **描述性统计**：自动生成医学论文标准的 Table 1（基本信息表）。
- **假设检验**：包含 T 检验、卡方检验及单因素方差分析（ANOVA）的 R 脚本。
- **回归模型**：Logistic 回归分析在疾病风险预测中的应用。

### 3. 📈 医学可视化 (Data Visualization)
- **基础图表**：使用 `ggplot2` 绘制箱线图、散点图。
- **专业图表**：
  - **生存曲线 (Kaplan-Meier)**：用于分析患者预后。
  - **森林图 (Forest Plot)**：用于 Meta 分析或多因素回归结果展示。

---

## 🛠 工具栈
- **语言**：R (v4.0+)
- **IDE**：RStudio
- **核心包 (Libraries)**：
  - `tidyverse` (数据处理核心)
  - `ggplot2` (高级绘图)
  - `survival` (生存分析)
  - `tableone` (快速生成基线特征表)

---

## 🚀 快速开始
1. 确保已安装 R 和 RStudio。
2. 克隆仓库到本地。
3. 打开 `.Rproj` 文件，运行 `scripts/` 文件夹下的分析脚本。
